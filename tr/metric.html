Metrik ifade çevirimi

Kullanım 
convert.Metric(veri, kaynak, hedef)
Parametreler:
Veri => Çevrilmesini istediğimiz değer
Kaynak => Değerin şu anki birimi
Hedef => Çıktı olarak almak istediğimiz birim
Kaynak ve hedef türleri:
  • cm => Santimetre
  • km => Kilometre
  • hm => Hektometre
  • m => Metre
  • mm => Milimetre
  • inch => İnç
  • feet => fit
  • yard => yarda
